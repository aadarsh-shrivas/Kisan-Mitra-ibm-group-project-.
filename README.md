# Kisan Mitra - Farmer's Crop Website

This project is a simple web application designed to help farmers by providing them with the latest information on weather, market prices for their crops, and guides for crop cultivation.

## Demo

The `demo` directory contains a sample website that demonstrates the features of this project. To run the demo, simply open the `index.html` file in your web browser.

## Features

*   **Weather Forecast:** Get real-time weather updates for your location.
*   **Mandi Prices:** Check the latest market prices for various crops in different markets.
*   **Crop Guides:** Access comprehensive guides for cultivating different crops.
